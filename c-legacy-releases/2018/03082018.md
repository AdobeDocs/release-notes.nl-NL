### [!DNL Experience Cloud] ID-service {#mcvid}

Verbeteringen en correcties voor de Experience Cloud ID-service.

* Het probleem waarbij de functie niet werkte op subdomeinen, is opgelost. *`resetBeforeVersion`* (CORE - 11390)
* Probleem verholpen waarbij de Experience Cloud ID-service foutief een AMCV-cookie zou instellen op domeinen zonder de regelafstandstip als er een regelafstandstip [!DNL  visitor.cookieDomain] is. Er wordt bijvoorbeeld een [AMCV-cookie](https://marketing.adobe.com/resources/help/en_US/whitepapers/cookies/cookies_mc.html) ingesteld [!DNL  sampledomain.com] naast [!DNL  .sampledomain.com]. (CORE - 10634)

Zie de productdocumentatie van [Experience Cloud ID Services](https://docs.adobe.com/content/help/nl-NL/id-service/using/home.html) .
